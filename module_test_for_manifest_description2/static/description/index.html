Customizable Table of Contents

Select which heading levels (H1, H2, H3, etc.) to include in the Table of Contents. Exclude specific headings that do not need to appear in the list. Enable automatic numbering for headings to provide a structured overview. Convert headings to HTML IDs for seamless in-page navigation.

Improved Navigation

Readers can quickly jump to desired sections of a blog post using the Table of Contents. The floating or inline display options make navigation smooth and intuitive.

SEO Optimization

The Table of Contents improves blog indexing by search engines, ensuring better ranking and visibility.
